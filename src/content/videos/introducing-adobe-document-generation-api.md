The first Paper Clips episode introducing Adobe Document Generation API. The video covers the core concept: author a Word template with special JSON tags, pass in a data payload, and get back a formatted Word or PDF document. It walks through the tagging syntax, the API call structure, and a live demo showing the output. A good starting point for anyone new to document automation with Adobe Document Services.
