A session from Adobe Developers Live (October 2021) introducing Adobe PDF Services API to developers. The talk covers the full range of PDF operations the API supports — creating PDFs from HTML or Office files, merging, splitting, OCR, compression, and more — and walks through the SDK setup and a first API call. A practical starting point for developers adding PDF capabilities to applications.
